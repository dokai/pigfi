Repos
-----

The official source code repository is 

https://github.com/pig-fi/pigfi

Running development server
--------------------------

Example::

    ~/google_appengine/dev_appserver.py src

Deploy
------

Example::
        
    ~/google_appengine/appcfg.py update src
